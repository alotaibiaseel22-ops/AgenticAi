# Agentic AI Project Supervisor

## Vision
An autonomous multi-agent system for university capstone teams that monitors project progress, identifies risks, and recommends actions to improve the likelihood of on-time delivery.

## Core Objectives
- Autonomous project monitoring
- Risk detection and analysis
- Actionable recommendations
- Dashboard-based visibility
- Multi-agent collaboration using CrewAI

## MVP Scope
- Create project
- Add team members
- Generate project plan
- Track progress manually
- Analyze project risks
- Generate recommendations
- Monitor status through dashboard

## Agent Architecture
1. Coordinator Agent
2. Planner Agent
3. Progress Agent
4. Risk Agent
5. Recommendation Agent

## Tech Stack
- Claude API
- CrewAI
- FastAPI
- React / Next.js
- PostgreSQL
